Manipulate a local wallet backed by `zcash_client_sqlite`

Usage: zcash-devtool.exe wallet [OPTIONS] <COMMAND>

Commands:
  init              Initialise a new light wallet
  init-fvk          Initialise a new view-only light wallet
  display-mnemonic  Decrypt and display the wallet's mnemonic recovery phrase, if any
  reset             Reset an existing light wallet (does not preserve imported UFVKs)
  import-ufvk       Import a UFVK
  upgrade           Upgrade an existing light wallet
  sync              Scan the chain and sync the wallet
  enhance           Ensure all transactions have full data available
  balance           Get the balance in the wallet
  generate-account  Generate a new account in the wallet
  list-accounts     List the accounts in the wallet
  generate-address  Generate a new address for an account in the wallet
  list-addresses    List the addresses for an account in the wallet
  derive-path       Derive key material at a particular path below the wallet seed
  list-tx           List the transactions in the wallet
  list-unspent      List the unspent notes in the wallet
  shield            Shield transparent funds received by the wallet
  propose           Propose a transfer of funds to the given address and display the proposal
  send              Send funds to the given address
  tree              Commands that operate directly on the note commitment trees
  help              Print this message or the help of the given subcommand(s)

Options:
  -w, --wallet-dir <WALLET_DIR>  Path to the wallet directory
  -h, --help                     Print help
